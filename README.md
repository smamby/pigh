# Proyecto integrador
## Gestion de Hoteles tipo booking  

### Instrucciones:  
1)  - Clonar el repositorio  
git clone https://github.com/smamby/pigh.git  
   
2)  - Copiar el .env en /  
   
3)  - Instalar las librerias y correr el backend     
npm install  
npm start
   
4)  - Abrir navegador
http://localhost:3001/
